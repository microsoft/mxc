# Support

## How to file issues and get help

This project uses [GitHub issues][gh-issue] to [track bugs][gh-bug] and [feature requests][gh-feature]. Please search the existing issues before filing new issues to avoid duplicates. For new topics, file your bug or feature request as a new issue.

For help and questions about using this project, please look at the [project README](./Readme.md), the [schema and backend documentation under `docs/`](./docs/), and our [Contributor's Guide](./CONTRIBUTING.md) if you want to work on MXC.

To report a security vulnerability, please follow the process in [SECURITY.md](./SECURITY.md) — **do not** open a public GitHub issue.

## Microsoft Support Policy

Support for MXC is limited to the resources listed above.

[gh-issue]: https://github.com/microsoft/mxc/issues/new/choose
[gh-bug]: https://github.com/microsoft/mxc/issues/new?assignees=&labels=Issue-Bug,Needs-Triage&projects=&template=Bug_Report.yml
[gh-feature]: https://github.com/microsoft/mxc/issues/new?assignees=&labels=Issue-Feature,Needs-Triage&projects=&template=Feature_Request.yml
